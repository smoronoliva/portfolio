5 · CSS Class Reference
ClassDescription
Window Chrome
.win95-windowBase window shell. Beige bg, navy border, rounded top corners (8px), drop shadow. Use as a flex column container.
.win95-dialogFixed-width modal variant (min 320px, max 420px). Same chrome as .win95-window but not draggable/resizable.
.title-barBlue gradient header row. Contains .title-bar-text + .title-bar-controls.
.title-bar-textBold 12px white text in the title bar. Truncates with ellipsis.
.title-bar-controlsFlex row for title bar action buttons (gap 3px).
.title-bar-btn22×21px rounded button on the title bar. Blue gradient by default.
.spa-btn-closeModifier on .title-bar-btn — applies red gradient. Also matches #dialog-close-btn.
.spa-btn-minimizeMinimize button on a .spa-window title bar (renders "_").
.spa-btn-maximizeMaximize button — CSS-drawn square icon via ::before/::after pseudo-elements.
.spa-btn-maximize.is-maximizedRestore state of maximize button — two overlapping squares.
.menu-barSecondary nav row below title bar. Beige bg, bottom border.
.menu-bar-itemClickable label in the menu bar. Highlights with --xp-highlight on hover.
.window-bodyScrollable content area. 8px padding, beige bg, flex: 1.
.status-barFixed-height row at window bottom. Beige bg, top border.
.status-bar-fieldIndividual cell inside .status-bar. Inset border, fills available width.
.win95-window.minimizedHides .window-body, .menu-bar, .status-bar — shows only title bar.
.spa-window.maximizedFills the desktop area (inset: 0) with no border-radius.
SPA Window System
.spa-windowAbsolutely-positioned draggable/resizable window inside #desktop-windows. resize: both.
.spa-title-barMarks a title bar as the drag handle in the window manager.
.photo-windowModifier for photo viewer windows — window-body is flex:1, padding:0, overflow:hidden.
.case-study-spa-windowModifier for the case study window — window-body is flex:1 flex-column, allowing slide panel to grow.
Content Panels
.panel-raisedBeige bg, steel border, subtle outer shadow. Used for secondary groupings on the beige surface.
.panel-sunkenWhite bg, steel border, inset shadow. Primary content container — bio, cards, job entries.
Controls
